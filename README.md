# Negative

## Build Process

1. Clone the project.
    ```
    git clone https://github.com/atdrago/negative.git
    ```
2. Move into the project.
    ```
    cd negative
    ```
3. Install modules.
    ```
    npm install
    ```
4. Build the app.
    ```
    grunt
    ```
5. Locate and copy `dist/Negative-darwin-x64/Negative.app` to your `Applications` folder
